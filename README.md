Employee Management System

This is a Windows Forms desktop application developed using C# and ADO.NET with a PostgreSQL database. 
The system provides full CRUD (Create, Read, Update, Delete) functionality for managing employee records.

Key Features:
- Employee record management
- DataGridView-based data display and interaction
- Auto-increment employee IDs
- Add, Edit, and Delete operations
- Export data to CSV/Excel
- PostgreSQL database integration using Npgsql

- Technologies Used:
- C#
- .NET Framework (Windows Forms)
- ADO.NET
- PostgreSQL
- Npgsql
- GitHub (Version Control)
